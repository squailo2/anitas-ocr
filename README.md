# Anita‚Äôs OCR

**Anita‚Äôs OCR** is a free, open-source web app that lets you extract text from images and PDFs directly in your browser.
Powered by [Tesseract.js](https://github.com/naptha/tesseract.js) and [PDF.js](https://mozilla.github.io/pdf.js/).  

---

##  Features :)
-  OCR Boxes using Tesseract.js
-  PDF support powered by PDF.js
-  Result Cleaner Filters
-  Multi-Page Batch Processing. Selections will be preserved across pages. The OCR will run only when you click the _batch OCR button_.
-  Dividers to crate paragraphs in the OCR box
-  OCR result tools. 
-  Works fully offline once loaded (libraries cached)
-  100% privacy-friendly ‚Äî processing stays in your browser
-  Simple, responsive interface
-  OCR Engine Provide a URL to your custom .traineddata file to improve accuracy on specific documents.
